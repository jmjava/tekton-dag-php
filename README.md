# tekton-dag-php

Sample PHP (Composer) app for [tekton-dag](https://github.com/jmjava/tekton-dag) pipeline stacks.

- **Build tool:** composer  
- **Runtime:** php  
- **PHP:** 8.3  

Use with stack app config: `tool: composer`, `runtime: php`, `build-command: "composer install --no-dev --optimize-autoloader"`.
